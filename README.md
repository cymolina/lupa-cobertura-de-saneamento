# lupa-cobertura-de-saneamento

Este projeto demonstra uma lupa de cobertura de saneamento, permitindo analisar:

Consumidores dentro de uma área móvel

Cobertura de água

Extensão de rede de água

Indicadores calculados dinamicamente conforme a lupa é movida

O mapa é totalmente interativo e utiliza dados vetoriais (GeoJSON) combinados com operações geométricas via Turf.js, além de visualização em Leaflet.

Este exemplo foi desenvolvido com fins didáticos, usando dados sintéticos de consumidores e dados reais da IDE-ES/CESAN para redes de distribuição de água.
